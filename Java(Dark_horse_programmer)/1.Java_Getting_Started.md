1. JVM(JAVA虚拟机)
2. 核心类库
3. 开发工具（javac编译;Java运行工具;jdb调试工具;jhat内存分析工具）

JRE（Java Runtime Environment Java运行环境）：

1. JVM(JAVA虚拟机)
2. 核心类库
3. 运行工具
